# 🤖 AI Next Word Prediction using LSTM

A Deep Learning-based Next Word Prediction web application built using **TensorFlow, Keras, LSTM, and Streamlit**. The model predicts the most probable next word based on the user's input sentence.

---

## 📌 Features

- 🤖 AI-powered Next Word Prediction
- 🧠 LSTM (Long Short-Term Memory) Model
- 🌐 Interactive Streamlit Web Application
- 📊 Confidence-based Prediction
- ⚡ Fast Inference
- 🎨 Modern & Responsive User Interface
- 📚 Natural Language Processing (NLP)

---

## 🛠️ Tech Stack

- Python
- TensorFlow
- Keras
- Streamlit
- NumPy
- Pickle

## ⚠️ Note

This model was trained using a fixed vocabulary of **8,978 words**.

- The model can predict only words available in the trained vocabulary.
- Unknown or unseen words may reduce prediction accuracy.
- Best results are obtained when the input is related to the training dataset.

**Vocabulary Size:** **8,978 Words**
